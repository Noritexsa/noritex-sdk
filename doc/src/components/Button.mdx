# Botones

## Uso básico

<Playground>
  <Button>Button</Button>
</Playground>

## Variantes
Usar elemento padre **Buttons** para agregar separación en un grupo de botones

<Playground>
	<Buttons>
	  <Button>Default</Button>
	  <Button primary>Primary</Button>
	  <Button white>White</Button>
	  <Button static>Static</Button>
	  <Button loading>Button</Button>
	</Buttons>
</Playground>

## Responsive
En resoluciones igual o menor a 768px de ancho, el botón se expande al 100%.

<Playground>
	<Button responsive>Button</Button>
</Playground>
